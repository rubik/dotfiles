## OS

ArchLinux + OpenBox 3.5.0 + Tint2-svn + Conky 1.9.0

## Necessary packages

Desktop Environment

* openbox
* obconf
* conky
* volumeicon
* tint2-svn (in AUR - install it with yaourt or manually by downloading
  PKGBUILD and building the pacman package)

Other dotfiles are from

* thunar
  * thunar-volman
* vim
* xorg
  * xorg-xinit
